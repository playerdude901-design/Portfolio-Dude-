MIS TRABAJOS

Soy desarrollador y dueño de PresetGold, un plugin para Adobe Premiere Pro que optimiza y acelera el flujo de trabajo de editores de video. Incluye una barra de búsqueda inteligente de Custom Presets y efectos de video y audio, permitiéndote encontrar y aplicar ajustes al instante sin perder tiempo navegando entre paneles.

DESCARGAR DE GITHUB

AurumGold es un programa para edición y conversión de formato horizontal a vertical sin necesidad de pasar por otros programas. Extrae clips en formato vertical con manejo de múltiples cámaras y formas de las mismas, optimizando tu flujo de trabajo para contenido vertical y redes sociales.

DESCARGAR DE GITHUB

CutterGold es un editor de clips en masa. Puedes marcar y extraer múltiples clips definiendo la entrada y salida de cada uno, creándolos simultáneamente sin perder tiempo en procesos manuales repetitivos.
